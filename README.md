## Run Locally

```bash
git clone https://github.com/AyushAgrawal-04/Ayush_Agrawal.git
cd Portfolio_Website
npm install
npm run dev
```

---

## EmailJS Setup

Create a template with:

```
{{user_name}}
{{user_email}}
{{message}}
```

Then add your credentials in `ContactMe.jsx`.

---

## Customize

Edit content in:
* `Hero.jsx`
* `Projects.jsx`
* `Qualifications.jsx`
* `Skill.jsx`
* `/public` (images & resume)
